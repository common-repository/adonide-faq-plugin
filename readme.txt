== Description ==
FAQ plugin is a quick, easy way to add an FAQs page.

- Add questions and answers.
- Customizing Style.
- Multiple Instances
- Updating the order of questions
- TinyMCE Boutton

== Installation ==

- In your Admin, go to menu Plugins > Add
- Search for `FAQ plugin`
- Click to install
- Activate the plugin
